Definition:
A mechanism for treating individual objects and compositions of objects in a uniform manner

Example:
Foo and List<Foo> have common APIs.
Graphical Objects example and Neurons example.

Notes:
1. Composition lets us make compound objects
2. Composite design pattern is used to treat both single and composite objects uniformly
3. Java supports container iteration with Iterable interface
4. A single object can masquerade as a collection by returning a single-element collection containing only this
